NodeXT web server

This module sets up the Express server used by NodeXT and registers the middleware and routes from all enabled extensions for it.

Functions

exports.createApplication(config)

Instantiate an Express server based on the configuration.

To enable SSL, provide something like the following configuration:

"server": {
  "hostname": "127.0.0.1",
  "port": 443,
  "privateKey": "privatekey.pem",
  "certificate": "certificate.pem"
},

Express view engines can be configured with:

"server": {
  ...
  "view": {
    "engine": "jade"
  }
}
